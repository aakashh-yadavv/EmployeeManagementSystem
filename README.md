# EmployeeManagementSystem <br>
Employee Management System ek web-based application hai jo company ke employees ke data ko manage karne ke liye use hota hai.<br> Isme admin employees ko add, update, delete aur view kar sakta hai.<br>
🚀 Features<br>
Add new employee<br>
Update employee details<br>
Delete employee<br>
View all employees<br>
Search employee <br>
Secure API handling<br>
Ye project basic CRUD operations ko samjhne ke liye banaya gaya hai aur real-world applications ke liye foundation provide karta hai.<br>
